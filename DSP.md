## 模块1：反距离加权插值 (IDW)

### 1. 任务目标

根据已知采样点内插出未知点 Q 的高程。

### 2. 数学原理

#### 2.1 距离计算

计算未知点 Q(x,y) 到各已知采样点 Pᵢ(xᵢ,yᵢ) 的平面距离：

$$d_i = \sqrt{(x - x_i)^2 + (y - y_i)^2}$$

#### 2.2 权值计算

$$W_i = \frac{1}{d_i^p}$$

其中 p 为权指数，通常取 p = 2。

#### 2.3 加权平均求值

$$H_Q = \frac{\sum_{i=1}^{n} W_i \cdot H_i}{\sum_{i=1}^{n} W_i}$$

其中：
- $H_Q$：待插值点的高程
- $H_i$：已知采样点的高程
- $W_i$：权值

### 3. 算法流程

```
1. 读取已知采样点数据（点号, X, Y, 高程）
2. 读取待插值点数据（点号, X, Y）
3. 对每个待插值点：
   3.1 计算到所有已知点的距离 dᵢ
   3.2 计算权值 Wᵢ = 1/dᵢᵖ
   3.3 加权平均求高程 H_Q = Σ(Wᵢ×Hᵢ)/ΣWᵢ
4. 输出插值结果
```

### 4. 实现要点

- **距离为零处理**：当待插值点与某已知点重合时，直接返回该已知点高程
- **权指数选择**：默认 p=2，可根据数据特性调整
- **搜索半径**：可设置最大搜索半径，减少计算量

### 5. 数据格式

**输入格式**：
```
# 已知采样点
# 格式: 点号 X Y 高程
P01 100.0 200.0 50.5
P02 150.0 250.0 52.3
P03 200.0 300.0 55.1

# 待插值点
# 格式: 点号 X Y
Q01 120.0 220.0
Q02 180.0 280.0
```

## 模块2：GPS高程拟合

### 1. 任务目标

利用已知点的高程异常值，拟合模型推求待定点高程异常。

### 2. 数学原理

#### 2.1 高程异常定义

$$\zeta = h - H$$

其中：
- $h$：GPS测得的大地高
- $H$：水准测量得到的正常高

#### 2.2 拟合模型

**平面拟合模型**：

$$\zeta = a_0 + a_1 X + a_2 Y$$

适用场景：小范围平坦地区（< 10km²）

**二次曲面拟合模型**：

$$\zeta = a_0 + a_1 X + a_2 Y + a_3 X^2 + a_4 XY + a_5 Y^2$$

适用场景：中等范围地区（10-100km²），地形有一定起伏

#### 2.3 最小二乘求解

建立误差方程：

$$V = AX - L$$

法方程：

$$A^T A X = A^T L$$

解：

$$X = (A^T A)^{-1} A^T L$$

### 3. 精度评估

**中误差**：

$$\sigma = \sqrt{\frac{\sum v^2}{n - k}}$$

其中 n 为已知点数，k 为参数个数。

### 4. 算法流程

```
1. 读取已知点数据 (X, Y, ζ)
2. 构建设计矩阵 A 和观测向量 L
3. 求解法方程 (A^T A)X = A^T L
4. 计算残差 V = AX - L
5. 评估模型精度 σ = √[Σv²/(n-k)]
6. 选择最优模型（σ最小）
```

## 模块3：时间系统转换

### 1. 任务目标

实现格里高利历（年月日）、儒略日（JD）及 GPS 时间的互转。

### 2. 数学原理

#### 2.1 公历转儒略日 (JD)

若月份 M ≤ 2，则：
- Y = Y - 1
- M = M + 12

计算儒略日：

$$JD = \lfloor 365.25Y \rfloor + \lfloor 30.6001(M+1) \rfloor + D + 1720981.5 + \frac{UT}{24}$$

其中：
- Y：年
- M：月
- D：日
- UT：世界时（小时）

#### 2.2 儒略日转GPS周/周内秒

GPS时间起点：JD₀ = 2444244.5（1980-01-06 00:00:00 UT）

$$Week = \lfloor \frac{JD - JD_0}{7} \rfloor$$

$$Seconds = (JD - JD_0 - Week \times 7) \times 86400$$

#### 2.3 GPS周/周内秒转儒略日

$$JD = JD_0 + Week \times 7 + \frac{Seconds}{86400}$$

#### 2.4 儒略日转公历

通过逆推算法将儒略日转换为年月日时分秒。

### 3. 算法流程

```
公历 → 儒略日:
1. 处理月份（若M≤2，Y=Y-1, M=M+12）
2. 计算 JD = ⌊365.25Y⌋ + ⌊30.6001(M+1)⌋ + D + 1720981.5 + UT/24

儒略日 → GPS时间:
1. 计算 GPS起点 JD₀ = 2444244.5
2. Week = ⌊(JD - JD₀)/7⌋
3. Seconds = (JD - JD₀ - Week×7) × 86400

GPS时间 → 儒略日:
1. JD = JD₀ + Week×7 + Seconds/86400

儒略日 → 公历:
1. 通过逆推算法计算年月日时分秒
```

### 4. 数据格式

```
# 公历时间
# 格式: 年 月 日 时 分 秒
2026 3 11 14 30 0.0

# 儒略日
2460745.104167

# GPS周和周内秒
# 格式: 周数 周内秒
2300 345678.5
```

## 模块4：多边形面积计算

### 1. 任务目标

计算由多个坐标点围成的多边形面积。

### 2. 数学原理

#### 2.1 鞋带公式

$$Area = \frac{1}{2} \left| \sum_{i=1}^{n} (x_i y_{i+1} - x_{i+1} y_i) \right|$$

其中 $x_{n+1} = x_1, y_{n+1} = y_1$。

#### 2.2 三角剖分法（辅助校验）

以多边形重心为原点，将多边形分解为 n 个三角形：

$$S = \sum_{i=1}^{n} S_{\triangle O P_i P_{i+1}}$$

单个三角形面积（海伦公式）：

$$S_{\triangle} = \sqrt{p(p-a)(p-b)(p-c)}$$

其中 $p = (a+b+c)/2$。

### 3. 算法流程

```
1. 读取多边形顶点坐标
2. 点序检查（确保顺时针或逆时针）
3. 鞋带公式计算面积
4. 三角剖分法验证（可选）
5. 输出面积结果
```

### 4. 精度控制

- 面积计算精度：10⁻⁶ m²
- 双重验证差异阈值：10⁻⁶ m²

## 模块5：坐标转换

### 1. 任务目标

空间直角坐标 (XYZ)、大地坐标 (BLH) 与站心坐标 (NEU) 互转。

### 2. 数学原理

#### 2.1 椭球参数 (WGS-84)

| 参数 | 符号 | 值 |
|------|------|-----|
| 长半轴 | a | 6378137.0 m |
| 扁率 | f | 1/298.257223563 |
| 第一偏心率平方 | e² | 2f - f² |

#### 2.2 XYZ → BLH

**经度**：

$$L = \arctan2(Y, X)$$

**纬度迭代**：

$$p = \sqrt{X^2 + Y^2}$$

$$\theta = \arctan\left(\frac{Z \cdot a}{p \cdot b}\right)$$

$$B = \arctan\left(\frac{Z + e'^2 b \sin^3\theta}{p - e^2 a \cos^3\theta}\right)$$

迭代直至 $|B_{new} - B_{old}| < 10^{-10}$

**大地高**：

$$N = \frac{a}{\sqrt{1 - e^2 \sin^2 B}}$$

$$H = \frac{p}{\cos B} - N$$

#### 2.3 BLH → XYZ

$$X = (N + H) \cos B \cos L$$

$$Y = (N + H) \cos B \sin L$$

$$Z = [N(1 - e^2) + H] \sin B$$

#### 2.4 XYZ → NEU

$$\begin{bmatrix} N \\ E \\ U \end{bmatrix} = R \begin{bmatrix} X - X_0 \\ Y - Y_0 \\ Z - Z_0 \end{bmatrix}$$

其中 R 为基于基准点 (B, L) 的旋转矩阵。

### 3. 精度控制

- 纬度迭代收敛阈值：10⁻¹⁰ rad
- 坐标转换精度：10⁻⁶ m

## 模块6：滑坡变形速度与应变计算

### 1. 任务目标

分析监测点随时间位移的速率及点对间的形变。

### 2. 数学原理

#### 2.1 变形速度

$$V = \frac{\sqrt{\Delta X^2 + \Delta Y^2 + \Delta Z^2}}{\Delta t}$$

其中 Δt 为实际观测间隔天数。

#### 2.2 相对应变

$$\varepsilon = \frac{S_{t+1} - S_t}{S_t}$$

其中 S 为两个监测点之间的空间欧氏距离：

$$S = \sqrt{(X_2 - X_1)^2 + (Y_2 - Y_1)^2 + (Z_2 - Z_1)^2}$$

### 3. 算法流程

```
1. 读取多期监测数据
2. 按监测点分组
3. 对每个监测点：
   3.1 计算各期位移量
   3.2 计算变形速度
4. 对每个监测点对：
   4.1 计算各期距离 S
   4.2 计算相对应变 ε
5. 生成分析报告
```

### 4. 数据格式

```
# 监测点时序数据
# 格式: 点号 期数 X Y Z
M01 1 3548000.123 526000.456 1200.000
M01 2 3548000.125 526000.458 1199.995
M02 1 3548200.234 526100.567 1210.000
M02 2 3548200.236 526100.569 1209.998
```
